# You can add comments like this

# This is helpful in seperating group sets like this:
# BRANCH
# Format GROUP_NAME GROUP_ELEMENT_1-GROUP_ELEMENT_2-GROUP_ELEMENT_3
# spaces between group elements are allowed
# One GROUP_NAME can be used more than once ex:

BRANCH BEQ-BNE
BRANCH BLT-BGE-BLTU-BGEU

JUMP JAL-JALR
LOAD LB-LH-LW-LBU-LHU
STORE SB-SH-SW
SHIFT SLLI-SRLI-SRAI-SLL-SRL-SRA
ALU ADDI   -SLTI-  SLTIU  -XORI-ORI-   ANDI-SLLI-SRLI-SRAI 
ALU ADD-SUB-SLL-XOR-SRL-SRA-OR-AND   
MULDIV MUL-MULH-MULHSU-MULHU-DIV-DIVU-REM-REMU
MULTI MUL-MULH-MULHSU-MULHU
DIVI DIV-DIVU
REMI REM-REMU

# Aproximate groups
XALU XADD - XSUB
XMULDIV XMUL-XMULH-XMULHSU-XMULHU-XDIV-XDIVU-XREM-XREMU    
 
# Group groups
XNALU XALU- ALU
XNMULDIV XMULDIV - MULDIV
ALUMULDIV MULDIV - ALU - XALU- XMULDIV


# Ending Is Done by ENDOFLUT comment
ENDOFLUT
